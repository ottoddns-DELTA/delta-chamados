# Usuarios e permissoes

## Perfis

### Admin

Pode:

- Abrir chamados.
- Editar chamados.
- Iniciar atendimento.
- Resolver chamados.
- Gerenciar usuarios.
- Ver logs e auditoria.
- Criar, editar e excluir condominios.

### Monitoramento

Pode:

- Abrir chamados.
- Editar chamados.
- Cadastrar e editar condominios.

Nao pode:

- Iniciar atendimento.
- Excluir condominios.
- Gerenciar usuarios.

### Tecnico

Pode:

- Ver chamados.
- Iniciar atendimento.
- Resolver chamados com descricao do que foi feito.

## Usuarios criados

Senha provisoria usada na carga inicial:

```text
Delta@2026
```

Usuarios:

```text
otto.chamados       admin
alan.chamados       admin
joao.chamados       tecnico
joazinho.chamados   tecnico
```

## Recomendacao

Trocar senhas provisorias depois do primeiro acesso.

O admin pode editar usuario e trocar senha em:

```text
Administracao > Usuarios
```
